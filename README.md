# Hybrid-Powered-Dual-Brush-Dry-Cleaning-System-for-Enhancing-Solar-Photovoltaic-Panel-Efficiency
A low-cost waterless solar panel cleaning robot uses a nylon brush and microfiber roller to remove dust. It moves over panels without water or air, powered by a rechargeable solar-charged battery. The lightweight, modular design improves cleaning efficiency for homes and small solar farms.
